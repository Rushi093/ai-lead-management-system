# AI Lead Management System

## Project Overview
This project is an AI-powered Lead Management System that helps businesses capture, qualify, assign, and respond to customer leads using AI automation.

## Features
- Lead capture system
- AI lead qualification
- Hot/Warm/Cold lead classification
- Spam lead detection
- AI chat assistant
- Developer assignment
- Lead dashboard
- Conversation history

## Tech Stack
- Backend: Python, FastAPI
- Frontend: Streamlit
- Database: SQLite
- AI API: Groq API
- API Documentation: Swagger UI

## Setup Instructions
1. Clone the repository
2. Create virtual environment
3. Install requirements
4. Add Groq API key in .env file
5. Run FastAPI backend
6. Run Streamlit frontend
